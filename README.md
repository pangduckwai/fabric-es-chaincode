### fabric-es-chaincode

Create the docker image for Hyperledger Fabric v2.x external chaincode. This is currently 
used `fabric-es` and `fabric-cd` repos. 

**Versions**  

- Fabric v2.2.0
- Typescript v4.0.x
- Jest v26.x
